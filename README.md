# Introduction
The code implements the Dijkstra's algorithm using a priority queue and a graph with adjacency list. 

# Requirements
* CMake > 2.8

# Usage
* mkdir bin
* mkdir build
* cd build
* cmake ..
* make
* cd ..
* Run ./bin/dijkstra ./data/graphy_connectivity_file.txt, e.g., ./bin/dijkstra ./data/connectivity.txt
